1. INTRODUCTION

The FileUpload Component contains a set of Java classes that provide helper methods for handling RFC 1867. The following classes are included:

  • FileUpload - High-level class for parsing the request.
  • MultipartStream - Low-level processing of multipart data streams
  • FileItem - This interface represents a file received using multipart/form-data POST request
  • DefaultFileItem - Implementation of FileItem
  • FileUploadException - Exception thrown if processing the request data fails.

2. DEPENDENCIES

The FileUpload component is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.2 or later)
  • Java Servlets (Version 2.2 or later)
  • JUnit Testing Framework (Version 3.8.1 or later) - for unit tests only, not required for deployment
