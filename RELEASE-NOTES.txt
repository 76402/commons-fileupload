INTRODUCTION
============

This document contains the release notes for this version of the Commons
FileUpload component, and highlights changes since the previous version.

This is the first major release of Commons FileUpload. Documentation for this
release is available on the FileUpload web site:

    http://jakarta.apache.org/commons/fileupload/
	
There have been no functional changes since FileUpload 1.0 Release Candidate 1.
